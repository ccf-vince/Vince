Image Classification

1. 定義

給電腦一張圖片，自動告訴你這張圖的所屬類別

  • 影像預處理：就像拍照後修圖一樣，電腦會進行裁剪、去雜訊或增強對比，讓後續辨識更清楚。
  • 特徵提取：最關鍵的一步。電腦需要找出這張圖的特點，例如它的顏色、紋理、形狀或空間關係。
  • 分類器設計：最後根據這些特點，決定它是哪一類。

2. 傳統方法 vs. 深度學習

在過去（傳統機器學習），科學家必須手動告訴電腦要看哪些特徵（如特定邊緣），但這非常沒效率，且容易被背景雜訊干擾。

現在主流的是深度學習，像人類的大腦一樣，透過多層網路結構，一層一層地自動提取數據特徵，讓特徵更明顯、更容易分類。

3. CNN / DBN

卷積神經網路 (CNN)

CNN 是專門為處理影像設計的，它模仿人類視覺系統的工作方式，不斷進行 convolution 和 subsampling。

  • 卷積層 (Convolutional Layer)：像是一個過濾器，在影像上滑動來尋找特徵。會生成多張 feature maps。
  • 池化層 (Pooling Layer)：縮小影像數據，只保留重要資訊，加快運算速度。
  • 權重共享：減少參數數量，大幅提升計算效率。

最後透過全連接層完成分類。

深度信念網絡 (DBN)

DBN 是由多個 RBM 堆疊而成，透過「預訓練」與「微調」進行學習。相較 CNN，較能處理無標籤資料，但不專精影像。

比較表

特性 DBN CNN
主要架構 RBM 堆疊 卷積層 + 池化層 + 全連接層
訓練方式 預訓練 + 微調 反向傳播
特徵提取 單一特徵 多層次特徵
輸入處理 需手動特徵 直接處理影像
核心優勢 層次化抽象 權重共享、高容錯

結論：CNN 準確度遠高於 DBN，因為能多層次捕捉影像特徵。

一、CNN 設計動機

  • 全連接網路參數過多，容易過擬合
  • 利用影像局部特徵與重複模式簡化模型

二、卷積層

  • Receptive Field：每個神經元只關注局部區域
  • Parameter Sharing：同一濾鏡掃描整張圖

三、池化 (Pooling)

  • Max Pooling：選取區域最大值
  • 減少運算量
  • 現代可選擇不使用

四、CNN 架構

  1. 輸入：影像 Tensor（寬 × 高 × 通道）
  2. 處理：卷積層 + 池化層
  3. 輸出：Flatten + 全連接層 + Softmax

五、應用案例：AlphaGo

  • 將棋盤視為影像
  • 48 通道特徵
  • 不使用池化層

六、CNN 局限性

  • 對旋轉、縮放敏感
  • 需透過 Data Augmentation 解決
